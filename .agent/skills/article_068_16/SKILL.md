# 第六十八条の十六 （変更の届出）

> 第六十八条の十六 認証型式部材等製造者は、第六十八条の十一第二項の国土交通省令で定める事項に変更（国土交通省令で定める軽微なものを除く。）があつたときは、国土交通省令で定めるところにより、その旨を国土交通大臣に届け出なければならない。
