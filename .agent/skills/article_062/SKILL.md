# 第六十二条 （屋根）

> 第六十二条 防火地域又は準防火地域内の建築物の屋根の構造は、市街地における火災を想定した火の粉による建築物の火災の発生を防止するために屋根に必要とされる性能に関して建築物の構造及び用途の区分に応じて政令で定める技術的基準に適合するもので、国土交通大臣が定めた構造方法を用いるもの又は国土交通大臣の認定を受けたものとしなければならない。
