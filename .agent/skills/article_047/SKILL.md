# 第四十七条 （壁面線による建築制限）

> 第四十七条 建築物の壁若しくはこれに代る柱又は高さ二メートルをこえる門若しくはへいは、壁面線を越えて建築してはならない。ただし、地盤面下の部分又は特定行政庁が建築審査会の同意を得て許可した歩廊の柱その他これに類するものについては、この限りでない。 第三節 建築物の用途
