# 第七十七条の十一 （帳簿の備付け等）

> 第七十七条の十一 指定建築基準適合判定資格者検定機関は、国土交通省令で定めるところにより、建築基準適合判定資格者検定事務に関する事項で国土交通省令で定めるものを記載した帳簿を備え付け、これを保存しなければならない。
