# 第七十六条の二 （土地の共有者等の取扱い）

> 第七十六条の二 土地の共有者又は共同借地権者は、第七十条第三項（第七十四条第二項において準用する場合を含む。）、第七十五条の二第一項及び第二項並びに前条第一項の規定の適用については、合わせて一の所有者又は借地権者とみなす。
