# 第七十七条の五十三 （審査請求）

> 第七十七条の五十三 この法律の規定による指定認定機関の行う処分又はその不作為については、国土交通大臣に対し、審査請求をすることができる。この場合において、国土交通大臣は、行政不服審査法第二十五条第二項及び第三項、第四十六条第一項及び第二項、第四十七条並びに第四十九条第三項の規定の適用については、指定認定機関の上級行政庁とみなす。
