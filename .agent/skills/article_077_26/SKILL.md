# 第七十七条の二十六 （確認検査の義務）

> 第七十七条の二十六 指定確認検査機関は、確認検査を行うべきことを求められたときは、当該確認検査が大規模建築物に係るものである場合において当該指定確認検査機関が確認検査員を選任しないものであることその他の正当な理由がある場合を除き、遅滞なく、確認検査を行わなければならない。
